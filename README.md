# verilog-autoline

## Features

This Extension is help to edit verilog-file
Make align better at 'input', 'output', 'wire', 'reg' ....

## Keybinings

Defalut keybindings

* `ctrl+l`  - Auto Line
* `ctrl+alt+l` - Auto Line2
* `ctrl+f11` - Make VIM folding code automatically
